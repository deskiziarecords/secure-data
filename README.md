# Data Security Platform with SynthFuse

## Overview
A comprehensive data security platform enhanced with Synthetic Data Fusion, AI-Augmented Security, and Intelligent Orchestration.

## Key Features
- **DSPM (Data Security Posture Management)**: Continuous discovery, classification, and protection of sensitive data
- **DLP (Data Loss Prevention)**: Advanced data protection and leakage prevention
- **SynthFuse**: Synthetic data generation and AI-powered security orchestration layer

## Architecture
For detailed architecture, see [ARCHITECTURE.md](ARCHITECTURE.md)

## Quick Start
1. Clone the repository
2. Run setup scripts: `./scripts/setup/setup_environment.sh`
3. Deploy core services: `make deploy-core`
4. Deploy SynthFuse: `make deploy-synthfuse`
